FixNow

A MERN stack application that helps users find nearby mechanics in real-time during vehicle breakdowns. 
The platform uses Google Maps, Geolocation, Socket.IO, and MongoDB to connect users with mechanics instantly.

**Features**

- User Authentication
- Mechanic Registration
- Real-Time Service Requests
- Google Maps Integration
- Nearby Mechanic Search
- Route Navigation
- Request History
- Earnings Dashboard
- Review System
- Twilio SMS Notifications
- Razorpay Payment Integration

**Tech Stack**

Frontend:
- React.js
- React Router
- Socket.IO Client
- Google Maps API

Backend:
- Node.js
- Express.js
- MongoDB
- Socket.IO

Services:
- Twilio
